## 二、P0-1：搜索框跳动（根因 + 修复方案）

### 根因
`ui_common.py` 第 95-97 行：
```css
.search-inline { width: 150px; transition: all .15s; }
.search-inline:focus { width: 200px; }  /* ← 这里！focus 时突然变宽 50px */
```
输入框变宽 → 整个 `search-inline-group` 变长 → 按钮被挤到右边 → 用户点空。

### 修复方案
**去掉 width 跳变**，输入框固定宽度或用 flex 自适应：
```css
.search-inline { width: 180px; }           /* 固定，不跳 */
.search-inline:focus { width: 180px; }     /* focus 也不变 */
```
或者更彻底：让搜索框用 `flex: 1` 吃掉剩余空间，这样无论窗口多宽都不会跳。

### placeholder 改为
- 企业库：`"搜索企业 / 行业 / 关键词"`
- 资讯页：已经是 `"搜索标题/摘要/标签"`（没问题，不动）

**涉及文件**：`ui_common.py`（CSS）+ `gen_enterprise.py`（placeholder 文案）

这是你最在意的问题。我先把 298 家全部列出来，分成几类让你判断：

### 5.1 真正的行业媒体（约 5-10 家）

- 孝爱宝 — 同上

### 5.2 明显不是媒体的（占绝大多数 ~280+ 家）
以下企业被打上"行业媒体"标签是**错误的**，它们应该归入自己的真实业务类别：

**科技巨头（不该有"行业媒体"标签）：**
Google, IBM, Netflix, Airbnb, Salesforce, Coinbase, IAC, GE Healthcare

**金融巨头：**
美国银行, 摩根大通, 富达投资, 保德信金融, 美林证券

**医疗/保险巨头：**
强生, 梅奥诊所, Humana, UnitedHealth, Aetna, Cigna, 凯撒医疗, CVS Health, UnitedHealth Group

**消费品牌：**
Lululemon, OXO, Huel, GoodRx, Steadiwear, Nobel Hygiene

**汽车/制造：**
宝马, 大和房屋工业(Daiwa House), 积水房屋(Sekisui House)

**真实银发经济创业公司（约 200+ 家，各有所属赛道）：**
Eargo（助听器）、Nobi（智能硬件/跌倒检测）、ŌURA（智能戒指）、Steadiwear（辅具）、Jellydrops（认知症零食）、joyforall（陪伴机器人）、Stitch（社交平台）、GetSetUp（老年文娱）、Silvernest（养老社区）、GoGoGrandparent（出行）、Silver Bills（养老金融）、True Link（养老金融）……等等等等

### 5.3 根因分析
为什么会有 298 家"行业媒体"？回溯 `_rebuild_tags.py` 的逻辑：
1. 这些企业在原始数据中的 `category_l1` 或 `category_l2` 包含"行业媒体"
2. 上轮 BRAND_OVERRIDE 只处理了 **44 家**（足力健→鞋服、瑞贝卡→个护 等）
3. 剩余 **254 家** 没有被覆盖到 → 继续保留"行业媒体"标签

换句话说：BRAND_OVERRIDE 是"白名单纠错"模式（只改我知道的），但没有对剩余的 254 家做批量重新判定。

### 5.4 修复方案
**不能再用"白名单逐个纠错"模式了**，要改成"规则批量重判"：

**Step 1**：定义"什么才算真行业媒体"的硬标准：
- 主营业务 = 内容生产 / 资讯聚合 / 行业研究发布（不是"顺便有个公众号"）
- 收入来源 = 广告 / 会员订阅 / 峰会（不是卖产品/服务的公司）

**Step 2**：对当前 298 家逐条跑规则判定：
- 符合硬标准 → 保留"行业媒体"
- 不符合 → 删除"行业媒体"标签，**保留其其他正确标签**（这些企业大多已经有正确的其他标签了，比如 Eargo 已经有"助听器"、Nobi 有"智能硬件/健康监测"）

**Step 3**：预计结果：
- 真行业媒体：**~10 家以内**
- 被错误标记的：**~288 家** → 删掉"行业媒体"后各自归位到已有标签

> **注意**：这 288 家企业本身不会丢——它们还有其他正确的标签（健康管理/智能硬件/居家护理 等），只是多了一个错误的"行业媒体"。
